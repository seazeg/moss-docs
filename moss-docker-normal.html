Docker 各个镜像方式单独部署 ​

包含 Moss 应用镜像 + mysql 镜像 + redis 镜像

创建 Docker 自定义网络 ​

shell
docker network create bridge

MySql 镜像部署 ​

TIP

mySql 要求版本 5.7

拉取镜像

shell
docker pull mysql:5.7

启动容器

shell
docker run -itd --name mysql -p 3307:3306 -e MYSQL_ROOT_PASSWORD=WZiBbvYPwoAQLGB50QbV --restart=on-failure --network bridge mysql:5.7

Redis 镜像部署 ​

拉取镜像

shell
docker pull redis

启动容器

shell
docker run -itd --name redis -p 6379:6379 --restart=on-failure --network bridge redis

Moss-Server 镜像部署 ​

加载镜像 tar

shell
docker load -i moss-server-lastest.tar

启动容器

shell
docker run -itd --name moss-server -p 7002:7002 -v /docker/moss/server/logs:/usr/moss/app/logs --restart=on-failure --network bridge  moss-server-lastest
